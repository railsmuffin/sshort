# Sshort

Initiate an SSH connection to your Capistrano configured server

## Installation

```ruby
gem 'sshort'
```

And then execute:

```
bundle
```

Or install it yourself as:

```
gem install sshort
```
    
## Configuration

Add the following line to `Capfile`

```ruby
require 'sshort'
```

## Usage

```
cap ssh
```
